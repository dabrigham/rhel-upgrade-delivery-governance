Deployment Notes

The `web/` folder is now a self-contained static bundle. It can be served directly as the site root after review and approval.

Local Preview

python -m http.server 8000 --bind 127.0.0.1 --directory web

Then open the local page in a browser. The bind option keeps the preview server on the local machine only.

Clean Public Export

The current local staging git history is not intended to be pushed publicly because local scanner terms previously existed in that history. When publication is approved, create a fresh export folder with no `.git` history, copy only publication-safe files, confirm the local scanner terms file is excluded, rerun verification, and initialize a new public git repository only after approval.

Recommended Hosting

  • Use a personal or Brigham Workflow Systems page as the polished front door.
  • Use GitHub as the technical evidence layer.
  • Use LinkedIn Featured for distribution after publishing.

Publication Gate

Do not publish or deploy until the sanitized content is reviewed and explicitly approved.
